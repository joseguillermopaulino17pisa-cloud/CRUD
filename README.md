# CRUD de Tareas

## Descripción
Aplicación de tareas con login y autenticación usando JWT.

## Tecnologías
- Node.js
- Express
- SQLite
- HTML, JavaScript

## Cómo ejecutar

1. Ir a backend:
   cd backend
   npm install
   node server.js

2. Abrir frontend/login.html en el navegador

## Funciones
- Registro
- Login
- Crear tareas
- Ver tareas
- Editar tareas
- Eliminar tareas
